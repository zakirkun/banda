DOM-first by design

Banda renders directly to the DOM. What you write is what runs.

Fine-grained reactivity means only what changes, updates. No virtual diffing. No hidden complexity. 

const count = createState(0);

Card({
  title: "Fine-grained updates",
  children: [
    span({ text: `Count: ${count.get()}` }),
    Button({
      label: "Increment",
      onClick: () => count.set(n => n + 1)
    })
  ]
});
