DOM-First by Design

Banda renders directly to the DOM. No virtual layer between you and your UI. What you write is what runs.

Fine-grained reactivity means only what changes, updates. No diffing. No hidden complexity. Pure control. 

const count = createState(0);

Card({
  children: [
    span({ text: `Count: ${count.get()}` }),
    Button({
      label: "Increment",
      onClick: () => count.set(n => n + 1)
    })
  ]
});
